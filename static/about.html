About the Author

Hi Everyone, My name is Lalit Kumar and welcome to this project!

I am a DevOps enthusiast passionate about cloud-native technologies, Kubernetes, and infrastructure automation.

This project demonstrates an end-to-end GitOps deployment of a Go web application on Amazon EKS using ArgoCD, Helm, Docker, Kubernetes, NGINX Ingress Controller, and AWS Load Balancer.

Tech Stack used in this project:

- Docker & Docker Hub (Containerization)

- Amazon EKS (Managed Kubernetes)

- Helm (Kubernetes Package Manager)

- ArgoCD (GitOps Continuous Delivery)

- NGINX Ingress Controller

- GitHub Actions (CI/CD Pipeline)

Feel free to explore the repository and reach out on GitHub.
